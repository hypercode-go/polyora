
                              POLYORA
                              =======

Copyright (c) 2010 Julien Pilet <julien.pilet a calodox.org>

Polyora is a library to retrieve and track multiple objects from a video
stream.

This library was developed between 2008 and 2010 by Julien Pilet, at Keio
University, in Hideo Saito's laboratory.

Installation and usage instructions are available online:
http://www.hvrl.ics.keio.ac.jp/~julien/polyora/

Instructions can also be generated using Doxygen.
